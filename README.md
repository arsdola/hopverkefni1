# Hópverkefni 1 vefforritun 1 2019

# Uppsetning á verkefni:

Til að setja upp verkefnið:
```sh
npm install
npm run sass
```

Til að keyra verkefnið:
```sh
npm run sass
npm run dev
```

til að linta verkefnið:
```sh
npm run lint
```
